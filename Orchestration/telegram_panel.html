Chat Whitelist

Every chat that messages the bot is recorded here. Whitelist the chats that should be allowed to interact with the agent. Revoke at any time. The admin chat is always allowed automatically.

Message History

The last 200 messages per chat are kept in Redis and ingested into the tg.messages dataset on the data fabric, where they are indexed for vector and graph queries alongside everything else.

Send Message

Send an ad-hoc message from the bot to any chat — useful for testing, alerts, or quick interactions outside the polling loop. Broadcast sends the same message to every whitelisted chat.

Event Bridge

Forward selected events from the orchestrator's vera:events stream to a Telegram chat. Useful for getting pinged when a long-running DAG finishes, a research job completes, or a capability errors out.

Match by prefix — dag matches every DAG event, cap.error only matches errors.
